## Donator Guidelines üéÅ

When setting point values, consider:

1. **Original Retail Value**
   - Under $10 ‚Üí 5-10 points
   - $10-$25 ‚Üí 10-20 points
   - $25-$50 ‚Üí 20-35 points
   - $50-$100 ‚Üí 35-50 points
   - $100+ ‚Üí 50+ points

2. **Condition**
   - New in box: Full value
   - Like new: 80-100% of value
   - Gently used: 60-80% of value
   - Used but good: 40-60% of value

3. **Age Appropriateness**
   - Is it complete?
   - Are there missing pieces?
   - Is it safe for children?

4. **Seasonality**
   - Christmas-themed toys might be valued slightly higher
   - Consider if it's age-appropriate gift

## Admin Point Management üëë

### When to Adjust Points:

**Increase Points:**
- Family has more children than initially assessed
- Special circumstances (birthdays near Christmas)
- Returned toys (points are refunded automatically)
- Community donations allow for more allocation

**Decrease Points:**
- Accidental over-allocation
- Family situation changed

### How to Adjust:
1. Go to Admin Dashboard
2. Click "Users" tab
3. Find the receiver
4. Edit the number in the "Points" column
5. Press Enter or Tab to save

The system will automatically calculate:
- Total Points (what you set)
- Used Points (points spent on toys)
- Available Points (remaining to spend)

## Point Transactions üí≥

### Adding Toy to Sack:
- Points are **immediately deducted**
- Toy becomes **unavailable** to others
- Transaction is recorded

### Removing Toy from Sack:
- Points are **automatically refunded**
- Toy becomes **available** again
- Can be claimed by another receiver

## FAQs ‚ùì

**Q: What if I run out of points?**
A: Contact an admin. They may be able to allocate more based on availability.

**Q: Can I save points for later?**
A: Yes, but toys go fast during Christmas season!

**Q: What if a toy I want costs more than my remaining points?**
A: Choose a different toy or ask admin about possible reallocation.

**Q: As a donator, what if I set the wrong point value?**
A: You can edit your toy's details before it's claimed.

**Q: Can receivers trade points with each other?**
A: No, points are non-transferable. Only admins can reallocate.
